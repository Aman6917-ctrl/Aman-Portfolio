# 🚀 Aman Verma — Portfolio

A modern, award-worthy personal portfolio built with **HTML**, **CSS**, **JavaScript**, **GSAP**, and **Vite**. Features smooth page transitions, scroll-triggered animations, interactive project showcase, and a fully functional contact form.

🌐 **Live Preview**: _Coming Soon_

![Portfolio Preview](/public/images/hero/image-1.jpg)

---

## ✨ Features

- **Smooth Page Transitions** — GSAP-powered cinematic transitions between pages
- **Scroll Animations** — Dynamic content reveals & parallax effects using ScrollTrigger
- **Interactive Project Showcase** — Horizontal scroll gallery with hover effects
- **Working Contact Form** — Powered by Web3Forms, submissions go directly to email
- **Responsive Design** — Optimized for mobile (≤1000px) and desktop
- **Smooth Scrolling** — Lenis-based buttery smooth scroll experience
- **Custom Typography** — Three custom typefaces (Rader, Formula Narrow, Supply Mono)

## 🚀 Getting Started

### Prerequisites
- **Node.js** v16+
- **npm** or **yarn**

### Installation

```bash
# Clone the repository
git clone https://github.com/Aman6917-ctrl/Aman-Portfolio.git

# Navigate to the project
cd Aman-Portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

The dev server will start at `http://localhost:5173`

### Available Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server with hot reload |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |
| `npm run host` | Dev server accessible via network |
